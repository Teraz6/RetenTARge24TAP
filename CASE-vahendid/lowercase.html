LOWER CASE-VAHENDID

Lower CASE-vahendid keskenduvad teostulese, kus mudelitest saab tegelik tarkvaratoode.

Kasutamine

Nad toetavad andmebaasi struktuuri genereerimist, koodi genereerimist, testide läbiviimist, koodi versioonihaldust, konfiguratsioonihaldust, pöördprojekteerimist jms.

Ma olen kasutanud:

Visual Studio Code

GIT

PyCharm


Kasutanud pole:



Rakendust kasutatakse nõudmiste, defektide, vigade ja testide haldamiseks.








Rakendust kasutatakse arendamiseks. Peamiselt kirjutatakse kood Javas ja arendatakse Java rakendusi. Samuti on võimalik kasutada teisi keeli, näiteks Fortran, JavaScript, Python jne. Rakendus võimaldab kasutada paljusi plug-in mis laiendab keele valikuid.
